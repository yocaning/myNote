# 贡献文章说明

* 我要如何贡献文章
  * 首先git clone当前 [项目](https://github.com/yocaning/myNotes) 至本地（建议顺便素质三连）
  * 创建个人文章分支（ex:yocan）
  * 完成个人文章撰写
  * 在SUMMARY.md 文件中加上你的目录，以及相应的文件
  * commit and push
  * 然后提merge request就可以了，合并完成之后就可以立即展示在网页了
